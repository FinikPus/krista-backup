**********


Тестирование
============

В проекте реализованы базы для юнит и функциональных тестов. На данный
момент они выступают для *smoke* тестирования.


Юнит тестирование
-----------------

Unittest был выбран для упрощения поддержки тестирования старых версий
python.

Запуск юнит тестов:

   cd KristaBackup/
   python3 -m unittest


Функциональное тестирование
---------------------------

Для запуска функциональных тестов используется Pytest. Он выступает
обёрткой над запуском сценариев в Docker. Сценарии находятся в
"test/scenarios".

Запуск функциональных тестов:

   pytest

Также, при запуске можно выбирать версию python для тестирования.
Пример:

   pytest -2    # python3.2
   pytest -24   # python3.2 и 3.4
   pytest       # python3.5, стандартно
   python --all # запуск для python3.2-3.5


Дополнительно
=============

При написании кода и тестов нужно учитывать, что поддержка должна быть
для всех версий python, начиная с 3.2.
